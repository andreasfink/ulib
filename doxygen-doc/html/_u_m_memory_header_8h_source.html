9 /*
10  UMMemoryHeader is a structure in memory which can be prepended to any buffer allocated
11  from C. This one allows these memory blocks to be accounted for in UMObjectStat.
12  Instead of calling malloc() you simply call ummalloc() and you get a pointer to your data.
13  you must call umfree() to free such a block.
14  the pointer is actually offset by the header so pointer - header size is the header
15 */
