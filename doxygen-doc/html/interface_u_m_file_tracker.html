Detailed Description

A debugging helper object to track usage of files. Useful if you run out of file descriptors and you cant find which file you opened where and forgot to close. This goes togehter with the macros from UMFileTrackingMacros.h:

TRACK_FILE_FOPEN(f,c) TRACK_FILE_FCLOSE(f) TRACK_FILE_DESCRIPTOR(f,c) TRACK_FILE_SOCKET(f,c) TRACK_FILE_PIPE(f,c) TRACK_FILE_PIPE_FLF(f,c,a,b,d) TRACK_FILE_CLOSE(fdes) TRACK_FILE_ADD_COMMENT_FOR_FDES(fdes,c) TRACK_FILE_ADD_COMMENT_FOR_FILE(f,c)
