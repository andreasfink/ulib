Detailed Description

The root object for ulib.

UMObject is a replacement for NSObject. It allows a log handler to be attached, getting instantiated from a config file and it has some debug variant UMObjectDebug which allow to trace where objects get allocated and deallocated and it has methods to run methods in background in another thread.
