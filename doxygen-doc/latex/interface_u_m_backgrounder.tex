\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_u_m_backgrounder}{U\+M\+Backgrounder}} is an object to have background tasks handled properly. 

To implement a working tasks, subclass it and have it override the method \char`\"{}work\char`\"{}. \char`\"{}work\char`\"{} should call the sleeper\textquotesingle{}s sleep function when it waits for something so it could exit its main thread on the specific signal. \char`\"{}work\char`\"{} is run constantly in a loop if a positive value is returned. on 0 it goes to sleep for a short while to call \char`\"{}work\char`\"{} again to see if there\textquotesingle{}s more work ot be done. On a negative value it quits the backgroud thread.

to fire up the background thread, call start\+Background\+Task. to shut it down, call shutdown\+Background\+Task

\mbox{\hyperlink{interface_u_m_backgrounder_with_queue}{U\+M\+Backgrounder\+With\+Queue}} is a subclass who overrides work with a method to handle individual work items through a queue where the work queue can be shared by multiple backgrounders.
