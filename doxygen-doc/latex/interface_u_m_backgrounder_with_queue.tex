\doxysubsection{Detailed Description}
\mbox{\hyperlink{interface_u_m_backgrounder_with_queue}{U\+M\+Backgrounder\+With\+Queue}} is an object to run tasks from a single queue. 

The queue can be unique to one background thread or multiple background threads can share the same queue. This is useful if run on multi C\+PU machines so work items would be stuffed into the queue and picked off the queue.

To have a proper task queue to throw work objects to it you should use \mbox{\hyperlink{interface_u_m_task_queue}{U\+M\+Task\+Queue}} which instantiates a certain amount of U\+M\+Backgrounder\+With\+Queue\+Object attached to a single queue which you then can stuff \mbox{\hyperlink{interface_u_m_task}{U\+M\+Task}} objects into to get executed.
