\doxysubsection{Detailed Description}
A N\+S\+Date which remembers its previous value and if it has been modified. 

\mbox{\hyperlink{interface_u_m_integer_with_history}{U\+M\+Integer\+With\+History}} is a object holding a N\+S\+Integer and its previous value. It can be used to hold data which is potentially modified at some point in time and then remember if it has been modified and what the old values are. Used for example in database access to only modify the fields which have changed (and if none has changed, not doing any query at all).
