\doxysubsection{Detailed Description}
The root object for ulib. 

\mbox{\hyperlink{interface_u_m_object}{U\+M\+Object}} is a replacement for \mbox{\hyperlink{class_n_s_object-p}{N\+S\+Object}}. It allows a log handler to be attached, getting instantiated from a config file and it has some debug variant U\+M\+Object\+Debug which allow to trace where objects get allocated and deallocated and it has methods to run methods in background in another thread.
