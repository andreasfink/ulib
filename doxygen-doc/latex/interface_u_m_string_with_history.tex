\doxysubsection{Detailed Description}
A N\+S\+String which remembers its previous value. 

\mbox{\hyperlink{interface_u_m_string_with_history}{U\+M\+String\+With\+History}} is a object holding a N\+S\+String and its previous value. It can be used to hold data which is potentially modified at some point in time and then remember if it has been modified and what the old values are. Used for example in database access to only modify the fields which have changed (and if none has changed, not doing any query at all).
