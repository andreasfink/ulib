\doxysubsection{Detailed Description}
An array who\textquotesingle{}s access is synchronized so it can be accessed from multiple threads. 

\mbox{\hyperlink{interface_u_m_synchronized_array}{U\+M\+Synchronized\+Array}} can be used like N\+S\+Mutable\+Array (but its not a subclass of it). To create a \mbox{\hyperlink{interface_u_m_synchronized_array}{U\+M\+Synchronized\+Array}} from a N\+S\+Array use the init\+With\+Array initializer. To create a N\+S\+Array/\+N\+S\+Mutable\+Array from \mbox{\hyperlink{interface_u_m_synchronized_array}{U\+M\+Synchronized\+Array}}, use mutable\+Copy method.
