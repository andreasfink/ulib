\doxysubsection{Detailed Description}
A dictionary who\textquotesingle{}s access is synchronized so it can be accessed from multiple threads. 

\mbox{\hyperlink{interface_u_m_synchronized_dictionary}{U\+M\+Synchronized\+Dictionary}} can be used like N\+S\+Mutable\+Dictionary (but its not a subclass of it). To create a \mbox{\hyperlink{interface_u_m_synchronized_dictionary}{U\+M\+Synchronized\+Dictionary}} from a N\+S\+Dictionary use the init\+With\+Dictionary initializer. To create a N\+S\+Mutable\+Dictionary from \mbox{\hyperlink{interface_u_m_synchronized_dictionary}{U\+M\+Synchronized\+Dictionary}}, use mutable\+Copy method.
