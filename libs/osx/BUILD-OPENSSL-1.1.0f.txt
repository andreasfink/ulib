# You should build openssl 1.1.0f under MacOS X as following

1. unpack the tar apply, the patch and change the directory

tar -xvzf openssl-1.1.0f.tar.gz
patch openssl-1.1.0f/crypto/bio/bss_dgram.c < openssl-1.1.0f.diff
cd openssl-1.1.0f


2. Configure and build openssl with SCTP support

./Configure darwin64-x86_64-cc sctp no-shared


3. install it into the binary directory

make DESTDIR=../openssl-1.1.0f-binary  install
