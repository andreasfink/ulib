# You should build openssl 1.1.0k under MacOS X as following

1. unpack the tar apply, the patch and change the directory

wget https://www.openssl.org/source/openssl-1.1.0k.tar.gz
tar -xvzf openssl-1.1.0k.tar.gz
patch openssl-1.1.0g/crypto/bio/bss_dgram.c < openssl-1.1.0g.diff

(change include  netinet/sctp to include /Library/Frameworks/sctp.framework/Headers/sctp.h)
cd openssl-1.1.0g


2. Configure and build openssl with SCTP support

./Configure darwin64-x86_64-cc sctp no-shared


3. install it into the binary directory

make DESTDIR=../openssl-1.1.0k-binary  install



openssl-1.1.0g-binary/usr/local/include/
